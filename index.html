Classificação das Soluções

As soluções são sistemas homogêneos (monofásicos) compostos por uma dispersão de partículas de dimensões inferiores a 1 nanômetro (1 nm = 10⁻⁹ m).

Quanto ao Estado Físico

  • Sólidas: Componentes distribuídos em rede cristalina. Ex: Latão (Cu + Zn) e Bronze (Cu + Sn).
  • Líquidas: Solvente líquido. Soluto pode ser sólido (NaCl + H₂O), líquido (Álcool + H₂O) ou gasoso (O₂ + H₂O).
  • Gasosas: Sempre homogêneas. Ex: Ar atmosférico filtrado.

Quanto à Natureza do Soluto

  • Soluções Moleculares: O soluto é molecular. Não conduzem eletricidade. Ex: Açúcar (C₁₂H₂₂O₁₁).
  • Soluções Iônicas: O soluto sofre dissociação ou ionização. Íons livres permitem a condução de eletricidade.

Classificação das Soluções (Curva de Solubilidade)

A solubilidade (S) é a quantidade máxima de soluto que pode ser dissolvida em uma quantidade fixa de solvente, sob determinadas condições de temperatura e pressão.

O Coeficiente de Solubilidade (C.S.)

  • Insaturado: Quantidade de soluto inferior ao C.S.
  • Saturado: Quantidade de soluto igual ao C.S.
  • Supersaturado: Sistema instável com excesso de soluto dissolvido.

Análise de Gráficos (Curvas)

  • Curvas Ascendentes: Processos endotérmicos (absorvem calor).
  • Curvas Descendentes: Processos exotérmicos (liberam calor).
  • Inflexões: Indicam mudança na hidratação do sal. Ex: Na₂SO₄ · 10H₂O para Na₂SO₄ anidro.

Unidades de Concentração

Convenção: índice ₁ para soluto, ₂ para solvente e sem índice para a solução total.

Concentração Comum: C = m₁ / V (g/L)
Molaridade: M = n₁ / V ou M = m₁ / (MM₁ · V) (mol/L)
Molalidade: w = n₁ / m₂(kg) ou w = m₁ / (MM₁ · m₂(kg)) (mol/kg)

Título, Fração Molar e Densidade

  • Título em Massa (τ): τ = m₁ / (m₁ + m₂)
  • Fração Molar (X₁): X₁ = n₁ / (n₁ + n₂)
  • Densidade (d): d = m_total / V (g/cm³ ou g/mL)

Relações entre Concentrações

Dedução para chegar em C = d · τ · 1000:

  1. Definições base: C = m₁ / V e τ = m₁ / m_sol.
  2. Isolando a massa do soluto: m₁ = τ · m_sol.
  3. Substituindo em C: C = (τ · m_sol) / V.
  4. Introduzindo a Densidade: Como m_sol = d · V, temos C = (τ · d · V) / V.
  5. Ajuste de unidades: Densidade (g/mL) convertida para Litros (1000 mL).

Resultado Final: C = d · τ · 1000
