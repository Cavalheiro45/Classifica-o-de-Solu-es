Classificação das Soluções

As soluções são sistemas homogêneos (monofásicos) compostos por uma dispersão de partículas de dimensões inferiores a 1 nanômetro.

Quanto ao Estado Físico

  • Sólidas: Componentes distribuídos em rede cristalina. Ex: Latão (Cu + Zn).
  • Líquidas: Solvente líquido. Soluto pode ser sólido, líquido ou gasoso.
  • Gasosas: Sempre homogêneas. Ex: Ar atmosférico filtrado.

Quanto à Natureza do Soluto

  • Soluções Moleculares: O soluto é molecular. Não conduzem eletricidade. Ex: Açúcar.
  • Soluções Iônicas: O soluto sofre dissociação ou ionização, permitindo a condução de eletricidade.

Curva de Solubilidade

A solubilidade (S) é a quantidade máxima de soluto dissolvida em uma quantidade fixa de solvente.

O Coeficiente de Solubilidade (C.S.)

Insaturado: < C.S.
Saturado: = C.S.
Supersaturado: > C.S. (Instável)

Unidades de Concentração

Concentração Comum: C = m₁ / V
Molaridade: M = m₁ / (MM₁ · V)
Molalidade: w = m₁ / (MM₁ · m₂_kg)

Relações entre Concentrações

Dedução para chegar em C = d · τ · 1000:

  1. C = m₁ / V e τ = m₁ / m_sol
  2. m₁ = τ · m_sol
  3. C = (τ · d · V) / V
Resultado: C = d · τ · 1000
